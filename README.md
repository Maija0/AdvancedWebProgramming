# Advanced Web Programming Project

This repository contains a full-stack web application built for an advanced web programming course. The project includes both a client-side frontend and a server-side backend.

## 📁 Project Structure
client/ → Frontend application (Vite + TypeScript)

server/ → Backend API (Node.js + TypeScript + Express)

## 🚀 Features
- User authentication using JWT
- Protected API routes using middleware
- Kanban board API structure
- MongoDB database connection (local)
- REST API built with Express

## 🛠️ Technologies Used
- Node.js
- Express
- TypeScript
- MongoDB (Mongoose)
- JWT (JSON Web Token)
- CORS
- dotenv
- morgan
